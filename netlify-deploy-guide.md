# Netlify拖拽部署指南 - 100%可行

## 为什么选择Netlify？
1. **无需任何设置** - 拖拽即部署
2. **完全免费** - 无任何费用
3. **立即公开** - 无需权限设置
4. **自动HTTPS** - 安全连接
5. **支持自定义域名** - 未来可绑定startupai.com

## 部署步骤（3分钟完成）

### 第一步：访问Netlify
1. 打开浏览器访问：**[https://app.netlify.com](https://app.netlify.com)**
2. 点击 "Sign up" 注册（可以用GitHub账号一键登录）
3. 完成注册/登录

### 第二步：拖拽上传
1. 登录后点击 "Add new site" > "Deploy manually"
2. 打开文件管理器，进入：`/home/brandonclaw/.openclaw/workspace/github-deploy`
3. 选择**所有文件**（Ctrl+A）
4. 拖拽到Netlify的 "Drag and drop your site output folder here" 区域
5. 等待上传完成（约30秒）

### 第三步：获取网站链接
上传完成后，Netlify会立即给你一个链接，如：
`https://ai-startup-diagnosis.netlify.app`

### 第四步：测试网站
1. 点击链接打开网站
2. 测试所有功能
3. 分享链接给我测试
