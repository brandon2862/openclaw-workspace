# 📋 产品规格书

## 产品名称
**AgentKerja** - AI Agent 团队为你的创业助力

> "Kerja" 在马来语中是"工作"的意思

---

## 🎯 一句话描述
一个通过 Telegram 就能使用的 AI agent 团队服务，帮助马来西亚创业者完成市场调研、内容创作、数据分析等任务。

---

## 👤 目标用户画像

### 主要用户
- **年龄:** 25-45 岁
- **身份:** 小微企业主、自由职业者、初创公司创始人
- **痛点:** 
  - 没有预算请专业团队
  - 不懂如何使用 AI 工具
  - 时间不够用，需要自动化

### 用户场景
1. **Ahmad** - 开了一家网店，需要写产品描述和社媒帖子
2. **Mei Ling** - 做咨询服务，需要做市场调研和竞品分析  
3. **Raj** - 刚创业，需要各种商业文档和计划书

---

## 🏗️ 核心功能

### 1. 自然语言任务
```
用户: "帮我调研马来西亚咖啡店市场，找出前5名竞争对手"
     ↓
Coordinator 分析 → Research Agent 执行 → 结果呈现
```

### 2. 多 Agent 协作
```
用户: "帮我写一篇 LinkedIn 帖子，关于AI在马来西亚的机遇"
     ↓
Research Agent (收集数据) + Content Agent (撰写) → 联合输出
```

### 3. 记忆与学习
- 记住用户的偏好和风格
- 保存任务历史，可追溯
- 学习用户的行业特点

### 4. 自动化工作流
- 定时任务 (每日新闻摘要、周报等)
- 触发式任务 (新订单 → 自动生成感谢邮件)
- 批量处理 (10 个产品 → 10 个描述)

## 💻 用户体验流程

### 新用户流程
```
1. 添加 Telegram Bot @AgentKerjaBot
2. 发送 /start 开始
3. 选择语言 (中/EN/BM)
4. 选择套餐
5. 开始使用！
```

### 日常使用
```
用户: 你好
Agent: 你好！我是你的 AI 助手团队 🚀
       今天需要帮你做什么？
       
       快捷命令:
       /research - 市场调研
       /content - 内容创作
       /data - 数据分析
       /help - 帮助

用户: 帮我写3个关于咖啡店的Instagram帖子
Agent: 好的！我来帮你创作 ☕
       
       请告诉我:
       1. 你的咖啡店叫什么名字？
       2. 主打什么特色？
       3. 目标受众是谁？
       4. 想要什么风格？(专业/轻松/文艺)

用户: 叫Cafe Melayu，主打本地咖啡，年轻人，轻松风格

Agent: 正在创作中...
       ━━━━━━━━━━━━ 100%
       
       ✅ 完成了！这是3个Instagram帖子:
       
       📱 帖子 1:
       "早安马来西亚！☀️ 来杯正宗Kopi O 开启新的一天 ☕
        #CafeMelayu #KopiMalaysia #本地咖啡"
       ...
       
       需要修改吗？或者还需要其他帮助？
```
