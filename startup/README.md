# 🚀 AI Agent 创业服务平台

> 帮助马来西亚创业者使用 AI agent 团队
> 基于 Claude Code 架构模式设计

---

## 📋 产品概述

### 目标用户
- 马来西亚创业者
- 想用 AI 但不知道如何开始的人
- 需要自动化但没有技术团队的小企业

### 核心价值
1. **零门槛** - 不需要懂技术，用自然语言描述需求
2. **团队协作** - 多个 AI agent 协同工作
3. **持续学习** - AI 记住你的偏好和历史

### 2. 服务类型

#### 📦 基础套餐 (RM 99/月)
- 每月 100 次 agent 任务
- 基础 Research + Content Agent
- 邮件支持

#### 📦 专业套餐 (RM 299/月)
- 每月 500 次 agent 任务
- 全部 6 种 Agent 类型
- 优先支持 + 定制工作流

#### 📦 企业套餐 (RM 999/月)
- 无限 agent 任务
- 专属 Agent 配置
- API 接入 + 优先支持

## 🔧 核心功能模块

### 1. 任务入口
```
用户输入 → 自然语言理解 → 任务分类 → 分配给合适的 Agent
```

### 2. Hooks 系统 (自动化流程)
```yaml
# 客户工作流配置
hooks:
  task_start:
    - notify: "任务已开始"
  task_complete:
    - save_to_memory: true
    - notify: "任务完成，点击查看结果"
  daily_summary:
    - generate_report: true
    - send_email: true
```

### 3. 记忆系统
```
客户记忆/
├── profile.md          # 客户基本信息
├── preferences.md      # 偏好设置
├── history/            # 任务历史
│   └── 2026-04.md
└── projects/           # 项目文件
    └── my-business/
```

## 💻 技术实现

### 技术栈
- **前端:** Telegram Bot / Web App
- **后端:** Python (FastAPI)
- **AI:** OpenClaw + 多模型
- **存储:** SQLite + 文件系统
- **部署:** Railway / Vercel

### 核心文件结构
```
startup/
├── README.md           # 本文件
├── product-spec.md     # 产品规格
├── agent-config.yaml   # Agent 配置
├── pricing.md          # 定价策略
├── landing-page/       # 落地页
├── api/                # API 服务
└── templates/          # 任务模板
```
