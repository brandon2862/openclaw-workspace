## 三层记忆体系

### L1: 工作记忆
- **位置:** 当前会话上下文
- **寿命:** 单次会话
- **内容:** 正在处理的任务、临时数据
- **清理:** 会话结束时自动清理

### L2: 短期记忆
- **位置:** `memory/YYYY-MM-DD.md`
- **寿命:** 30 天
- **内容:** 每日事件、决策、经验
- **维护:** 每日自动整理

### L3: 长期记忆
- **位置:** `MEMORY.md`
- **寿命:** 永久
- **内容:** 重要决策、核心知识、用户偏好
- **维护:** 每周回顾更新

## 每日进化任务

### 时间: 22:00

### 执行流程

#### 1. 回顾会话历史
```
会话历史 → 提取关键事件 → 识别重要决策
```

#### 2. 压缩整理记忆
```
每日文件 → 压缩冗余 → 提取精华 → 保存到短期记忆
```

#### 3. 分析总结
- **学会的新东西** - 今天学到的知识
- **犯的错误** - 今天犯的错误
- **解决方法** - 如何解决的

#### 4. 生成进化报告
```
分析结果 → 生成报告 → 提议可以固化的技能
```

#### 5. 发送报告
```
报告 → 发送给用户 → 等待反馈
```

## 技能固化标准

### 可以固化
- ✅ 重复使用 3 次以上
- ✅ 解决通用问题
- ✅ 可以被标准化
- ✅ 有长期价值

### 固化流程
1. 识别重复模式
2. 提取通用逻辑
3. 创建 skill 模板
4. 测试验证
5. 发布到 ClawHub (可选)

## 自我评估

### 评估维度
1. **任务完成率** - 完成的任务 / 总任务
2. **学习效率** - 新知识掌握速度
3. **错误率** - 错误次数 / 总操作
4. **用户满意度** - 用户反馈评分

### 评估频率
- **每日:** 简单统计
- **每周:** 详细分析
- **每月:** 全面评估
