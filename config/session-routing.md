- **版本:** 1.0

- **最后更新:** 2026-04-06

---

## 会话识别流程

### 第一步：上下文关联度检查
- **相关度高** → 保持现有会话和模型池
- **相关度低** → 开启新会话

### 第二步：任务分类与模型池选择
检查新会话的任务类型，根据关键词选择对应的模型池。

---

## 任务分类规则

### 🚀 高速池 (highspeed)
**关键词:** 简单、快速、心跳、检查、状态、天气、时间、问候
**特征:**
- 响应时间 < 3秒
- 简单问答
- 快速任务
- 心跳检查

**示例:**
- "今天天气怎么样？"
- "现在几点？"
- "系统状态检查"

---

### 🧠 智能池 (intelligent)
**关键词:** 分析、思考、决策、复杂、创意、设计、规划、建议
**特征:**
- 需要深度分析
- 多步骤任务
- 创意工作
- 复杂对话

**示例:**
- "帮我分析这个市场趋势"
- "设计一个产品方案"
- "这个决策应该怎么选？"

---

### 📝 文本池 (text)
**关键词:** 写作、文档、翻译、总结、编辑、校对、报告、邮件
**特征:**
- 文档处理
- 写作任务
- 翻译工作
- 内容总结

**示例:**
- "帮我写一封邮件"
- "翻译这段文字"
- "总结这篇文章"

---

### 👁️ 视觉池 (vision)
**关键词:** 图片、图像、视觉、截图、设计稿、照片、图表
**特征:**
- 图像分析
- 视觉理解
- 图片生成
- 设计审查

**示例:**
- "分析这张截图"
- "这个设计稿怎么样？"
- "生成一张图片"

---

## 自动选择规则

```yaml
routing:
  autoSelect: true
  defaultPool: "highspeed"
  fallbackPool: "highspeed"
  sessionPersistence: true
  maxSessionAge: 3600  # 1小时
```

## 特殊规则

### 代码任务
- **简单代码:** 高速池
- **复杂代码:** 智能池
- **代码审查:** 智能池
- **文档生成:** 文本池

### 创业相关
- **市场分析:** 智能池
- **商业计划:** 智能池
- **营销文案:** 文本池
- **产品设计:** 智能池

### 框架相关
- **配置管理:** 智能池
- **任务调度:** 高速池
- **记忆维护:** 高速池
- **进化报告:** 智能池
