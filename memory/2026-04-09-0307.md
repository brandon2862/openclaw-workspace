## 📋 总结

**FFF = Freakin Fast Fuzzy file finder**

| 特性 | 说明 |
|------|------|
| **速度** | <150ms 搜索 100k+ 文件 |
| **记忆** | frecency + history，记住你常用的文件 |
| **智能排序** | git状态、文件大小、定义匹配 |
| **容错** | typo-resistant，打错字也能搜到 |
| **MCP** | 可以集成到 Claude Code/OpenClaw |
