## 技术经验总结

### 浏览器自动化工具对比
1. **BrowserWing** - HTTP API服务，Web UI，MCP协议集成，可视化脚本录制
2. **Browser Use** - Python库+CLI，通过uv管理，持久会话，支持真实Chrome配置文件
3. **Chrome无头模式** - 简单截图，无需复杂配置

### 数据API访问经验
- 免费股票API大多需要注册和密钥
- 天气API (wttr.in) 无需密钥，简单可靠
- 飞书文档需要登录权限，公开访问受限
- 百度热搜可直接抓取，但数据格式有限

### 文件处理模式
- CSV作为中间格式，易于生成和转换
- Excel文件需要pandas库，安装依赖较多
- JSON适合结构化数据存储
- Markdown适合报告和文档
