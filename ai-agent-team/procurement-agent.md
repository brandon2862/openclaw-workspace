## 马来西亚家私供应链知识库

### 市场概况

#### 马来西亚主要家具产业集群

1. **Johor (柔佛州)** - 马来西亚最大的家具生产州
   - 特点：OEM/ODM 制造、金属和木制家具
   - 主要城市：Johor Bahru, Kluang

2. **Selangor (雪兰莪州)** - Klang Valley 核心区域
   - 特点：高端定制家具、展示厅集中
   - 主要城市：Klang, Shah Alam, Petaling Jaya

3. **Kuala Lumpur & Putrajaya** - 高端市场中心
   - 特点：设计师品牌、高端进口家具

4. **Melaka (马六甲州)** - 传统木家具
   - 特点：古典木家具、手工艺品

5. **Kedah (吉打州)** - 塑料家具制造
   - 特点：户外家具、预算型家具

#### Klang Valley 高端家具零售点

1. **IKEA Cheras** - 大众市场定位
2. **IKEA Mutiara Rini** - 中高端
3. **The Gardens Mall** - 高端进口家具
4. **Pavilion Kuala Lumpur** - 奢侈品牌
5. **1 Utama Shopping Centre** - 高端家居
6. **Sunsquare KL** - 本地高端品牌
7. **Mitsui Shopping Park** - 日式家具

### 目标客户群体画像

#### 高收入华人 (High-Income Chinese)
- 年龄：35-55岁
- 收入：月入 RM 15,000+
- 偏好：现代简约、中式古典混搭、进口品牌
- 购房区域：Klang Valley 高端公寓/别墅

#### 高收入马来族 (High-Income Malay)
- 年龄：30-50岁
- 收入：月入 RM 12,000+
- 偏好：现代马来风格、豪华欧式
- 购房区域：Klang Valley 高端住宅区

## 业务流程

### 1. 需求获取
- 来自销售转化 Agent 的客户需求
- 来自内容创作 Agent 的产品信息需求

### 2. 供应商匹配
- 根据需求筛选马来西亚/中国供应商
- 收集产品信息、价格、交期

### 3. 对比分析
- 多供应商对比表
- 性价比分析
- 推荐方案

### 4. 采购执行
- 样品确认
- 大货订单
- 物流安排

### 5. 交付跟进
- 物流追踪
- 到货验收
- 问题处理

## 常用话术

### 初次联系供应商
```
您好，我是[公司名]，正在为马来西亚高端室内设计项目寻找供应商。
我们专注于[产品类别]，请问贵司有生产吗？
请发送产品目录和报价单，谢谢。
```

### 询价
```
您好，请问这款[产品名]的批发价格是多少？
最小起订量(MOQ)是多少？
预计交期是多久？
能否提供样品？
```

### 物流咨询
```
请问你们提供马来西亚的物流服务吗？
运费如何计算？
从中国到巴生港(Port Klang)的运输时间？
