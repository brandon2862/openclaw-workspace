# AI Agent 团队 - 整体框架配置

## 团队架构

基于马来西亚高端室内设计项目的 AI Agent 团队，包含 9 个专业 Agent：

### 核心协调层
1. **AI CEO Agent** - 战略决策与统筹
2. **企业教练 Agent** - 团队优化与学习

### 业务执行层
3. **市场洞察 Agent** - 市场调研分析
4. **客户寻访 Agent** - 潜在客户获取
5. **内容创作 Agent** - 营销内容生成
6. **沟通协调 Agent** - 客户服务沟通
7. **销售转化 Agent** - 成交促进
8. **采购支持 Agent** - 供应链管理 (重点)
9. **售后服务 Agent** - 客户关系维护

---

## Agent 配置文件

### 1. AI CEO Agent
- **角色**: 团队战略决策者
- **模型**: xiaomi/mimo-v2-pro
- **职责**: 统筹所有 Agent 工作，设定目标，监控绩效
- **协作**: 协调所有 Agent

### 2. 企业教练 Agent
- **角色**: 团队优化与学习
- **模型**: xiaomi/mimo-v2-flash
- **职责**: 性能分析，问题诊断，优化建议
- **协作**: 辅助 AI CEO，优化所有 Agent

### 3. 市场洞察 Agent
- **角色**: 市场调研分析
- **模型**: xiaomi/mimo-v2-flash
- **职责**: Klang Valley 高端市场分析，客群画像
- **协作**: 为客户寻访、内容创作提供数据支持

### 4. 客户寻访 Agent
- **角色**: 潜在客户获取
- **模型**: xiaomi/mimo-v2-flash
- **职责**: 社交媒体分析，CRM 管理，自动化外联
- **协作**: 与市场洞察、沟通协调 Agent 配合

### 5. 内容创作 Agent
- **角色**: 营销内容生成
- **模型**: xiaomi/mimo-v2-flash
- **职责**: 文案、图像生成、视频脚本
- **协作**: 支持销售转化、市场推广

### 6. 沟通协调 Agent
- **角色**: 客户服务初步沟通
- **模型**: xiaomi/mimo-v2-flash
- **职责**: 智能问答、日程管理、CRM 更新
- **协作**: 贯穿客户获取到售后全流程

### 7. 销售转化 Agent
- **角色**: 成交促进
- **模型**: xiaomi/mimo-v2-pro
- **职责**: 需求分析、方案生成、合同辅助
- **协作**: 与采购支持、沟通协调 Agent 配合

### 8. 采购支持 Agent (重点)
- **角色**: 家私供应链管理
- **模型**: xiaomi/mimo-v2-flash
- **职责**: 
  - 马来西亚本地供应商筛选
  - 中国家私供应商管理
  - 产品信息收集
  - 价格谈判辅助
  - 物流协调
  - 库存管理
- **协作**: 支持内容创作、销售转化、售后服务

### 9. 售后服务 Agent
- **角色**: 客户关系维护
- **模型**: xiaomi/mimo-v2-flash
- **职责**: 智能客服、工单管理、满意度分析
- **协作**: 与销售转化、采购支持 Agent 配合

## 协作机制

### 中央协调平台
- AI CEO Agent 处于最高层级
- 企业教练 Agent 辅助优化
- 信息实时共享

### 触发机制
- Agent 完成任务后自动触发下一个相关 Agent
- 例：市场洞察 → 客户寻访 → 沟通协调 → 销售转化

### 反馈循环
- 每个阶段的结果反馈给上游 Agent 优化
- 企业教练监控整体效率

---

## 工作流程阶段

1. **战略规划** - AI CEO + 企业教练
2. **找客户** - 市场洞察 + 客户寻访
3. **精准定位** - 客户寻访 + 沟通协调
4. **内容制作** - 内容创作 + 采购支持
5. **跟进成交** - 销售转化 + 采购支持
6. **售后服务** - 售后服务 + 采购支持
