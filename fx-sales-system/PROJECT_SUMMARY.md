## 业务规则实现

### 报价规则
```
Option 1 = Agent SX Cost - 0.05
Option 2 = Agent SX Cost - 0.08  
Option 3 = Agent SX Cost - 0.10
Option 4 = Manual（用户输入）
```

### 计算规则
```
汇率换算：MYR = RMB / Sell Rate
利润计算：P&L = (Agent SX Cost - Sell Rate) × Source Amount
```

### 数据规则
- 选择客户自动带出相关资料
- 所有计算使用精确小数（4位汇率，2位金额）
- 数据验证和完整性检查
