# 汇率销售管理系统 - 用户手册

## 系统概述

汇率销售管理系统是一个内部使用的系统，用于管理汇率报价、客户资料、交易记录和利润报表。系统支持多用户角色，提供实时计算、数据分析和报表导出功能。

1. 打开浏览器，访问系统地址（如：http://localhost:5173）
2. 进入登录页面

- **管理员账号**: admin@example.com / admin123

- 勾选"记住我"选项，下次访问时自动登录

## 2. 仪表盘

### 2.1 功能概述
仪表盘显示系统的关键指标和最近活动，帮助用户快速了解业务状况。

### 2.2 主要组件
1. **今日统计**
   - 今日交易笔数
   - 今日利润总额

2. **本月统计**
   - 本月累计利润

3. **客户统计**
   - 客户总数

4. **最近交易**
   - 最近10笔交易记录
   - 显示客户、金额、利润等信息

5. **利润最高客户**
   - 本月利润最高的5个客户
   - 按利润降序排列

6. **月度趋势**
   - 最近6个月的交易趋势图
   - 显示交易笔数和利润变化

## 3. 客户管理

### 3.1 查看客户列表
1. 点击左侧菜单"客户管理"
2. 查看所有客户列表
3. 支持分页浏览

### 3.2 搜索客户
1. 在搜索框输入关键词
2. 支持按客户名、Marking、接收人、电话搜索
3. 实时显示搜索结果

### 3.3 新增客户（仅管理员）
1. 点击"新增客户"按钮
2. 填写客户信息：
   - 客户名称（必填）
   - Marking（可选）
   - 默认货币对（默认RMB/MYR）
   - 接收人信息
   - 银行信息
   - 备注
3. 点击"保存"

### 3.4 编辑客户（仅管理员）
1. 在客户列表中点击"编辑"按钮
2. 修改客户信息
3. 点击"保存"

### 3.5 删除客户（仅管理员）
1. 在客户列表中点击"删除"按钮
2. 确认删除
3. **注意**: 有交易记录的客户不能删除

## 4. 汇率交易

### 4.1 新增交易
1. 点击左侧菜单"新增交易"
2. 页面分为左右两部分：

#### 左侧：客户信息
1. **选择客户**
   - 从下拉列表选择客户
   - 或输入关键词搜索客户
   - 选择后自动带出客户资料

2. **客户资料显示**
   - Marking
   - 接收人信息
   - 银行信息
   - 默认货币对

#### 右侧：交易信息
1. **输入Agent SX Cost**
   - 输入Agent SX Cost数值
   - 系统自动计算3个报价选项：
     - Option 1 = Agent SX Cost - 0.05
     - Option 2 = Agent SX Cost - 0.08
     - Option 3 = Agent SX Cost - 0.10

2. **选择报价选项**
   - 点击Option 1-3卡片选择自动报价
   - 或选择Option 4手动输入汇率

3. **输入源金额**
   - 输入RMB金额
   - 系统实时计算：
     - 转换金额（MYR）
     - 利润（P&L）

4. **填写备注**
   - 输入交易备注信息

5. **保存交易**
   - 点击"保存"按钮保存交易
   - 点击"清空"按钮重置表单

### 4.2 交易计算规则
1. **汇率计算**
   ```
   MYR = RMB / Sell Rate
   示例：RMB 10,000 ÷ 1.4500 = MYR 6,896.55
   ```

2. **利润计算**
   ```
   P&L = (Agent SX Cost - Sell Rate) × Source Amount
   示例：(1.5000 - 1.4500) × 10,000 = 500.00
   ```

## 5. 交易记录

### 5.1 查看交易记录
1. 点击左侧菜单"交易记录"
2. 查看所有交易列表
3. 每页显示20条记录，支持分页

### 5.2 筛选交易
1. **按客户筛选**
   - 选择客户名称
   - 显示该客户的所有交易

2. **按日期筛选**
   - 选择开始日期和结束日期
   - 显示该时间范围内的交易

3. **按货币对筛选**
   - 选择货币对类型
   - 显示该货币对的交易

4. **组合筛选**
   - 支持多个条件组合筛选
   - 点击"筛选"按钮应用条件
   - 点击"重置"按钮清除筛选

### 5.3 查看交易详情
1. 在交易列表中点击交易记录
2. 查看交易详细信息：
   - 交易基本信息
   - 客户信息
   - 计算详情
   - 备注信息

## 6. 月度报表

### 6.1 查看月度报表
1. 点击左侧菜单"月度报表"
2. 选择年份和月份
3. 点击"查询"按钮

### 6.2 报表内容
1. **汇总统计**
   - 总交易笔数
   - 总源金额（RMB）
   - 总利润（P&L）

2. **按客户分组**
   - 每个客户的交易笔数
   - 每个客户的总利润
   - 每个客户的总金额

3. **按货币对分组**
   - 每个货币对的交易统计

4. **交易明细**
   - 该月的所有交易记录
   - 最多显示100条记录

### 6.3 导出Excel报表
1. 在月度报表页面
2. 点击"导出Excel"按钮
3. 浏览器自动下载Excel文件
4. 文件包含：
   - 交易明细工作表
   - 汇总统计工作表

## 10. 常见问题

### 10.1 无法登录
1. 检查用户名和密码是否正确
2. 确认账号是否被锁定
3. 联系管理员重置密码

### 10.2 交易保存失败
1. 检查必填字段是否填写完整
2. 验证数字格式是否正确
3. 检查网络连接是否正常
4. 联系系统管理员

### 10.3 报表数据不准确
1. 确认筛选条件是否正确
2. 检查数据录入是否有误
3. 联系管理员核对数据

### 10.4 系统响应缓慢
1. 检查网络连接
2. 减少同时打开的数据量
3. 联系管理员优化系统
