## 构建和打包系统

### 开发构建
```bash
# 开发服务器
npm run dev

# 构建主进程
npm run build:main

# 构建渲染进程
npm run build:renderer

# 完整构建
npm run build
```

### 生产打包
```bash
# Windows安装包
npm run dist:win

# 所有平台
npm run dist

# 便携版本
npm run pack
```
