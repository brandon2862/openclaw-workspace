## 🚀 使用指南

### 快速开始
```bash
# 1. 克隆项目
git clone <repository-url>
cd electron-vue-app

# 2. 安装依赖
npm install

# 3. 启动开发
npm run dev

# 4. 构建应用
npm run build

# 5. 打包安装包
npm run dist:win
```

### 开发工作流
1. **功能开发**：在 `src/renderer/views/` 添加新页面
2. **状态管理**：在 `src/renderer/stores/` 添加新 store
3. **系统集成**：在 `src/main/ipcHandlers.ts` 添加新 API
4. **类型定义**：在 `src/shared/types.ts` 添加类型
