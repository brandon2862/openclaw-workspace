# KV Premium Reno — 报价流程标准作业程序 (SOP)

> **版本：** 1.0
> **生效日期：** 2026年4月
> **适用范围：** 全体报价及销售团队
> **品牌承诺：** 透明报价，准时交付，让您的梦想空间零风险实现

---

## 📊 流程总览

```
┌─────────────┐    ┌─────────────┐    ┌─────────────┐    ┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│  1.客户咨询  │ → │  2.上门测量  │ → │  3.报价生成  │ → │  4.内部审核  │ → │  5.报价发送  │ → │  6.客户跟进  │
│   Inquiry    │    │   Measure   │    │   Quote     │    │   Review    │    │   Send      │    │   Follow-up │
└─────────────┘    └─────────────┘    └─────────────┘    └─────────────┘    └─────────────┘    └─────────────┘
     1-2天              1-3天             2-3天              1天               当天              持续跟进
```

**全流程目标时间：5-7个工作日完成报价**

## 📏 阶段二：上门测量（Site Measurement）

### 2.1 测量前准备

**出发前检查清单：**

- [ ] 确认预约时间、地址、客户联系方式
- [ ] 携带测量工具：
  - 激光测距仪
  - 卷尺（5米+）
  - 水平仪
  - 相机/手机（拍照用）
  - 笔记本+笔
  - 鞋套（进屋穿戴）
- [ ] 携带公司资料：
  - 名片
  - 作品集/案例照片
  - 公司介绍册
- [ ] 穿着公司制服

### 2.2 测量流程

**Step 1：到达与问候（5分钟）**
- 准时到达（提前5分钟）
- 自我介绍，出示名片
- 穿戴鞋套
- 简要说明测量流程

**Step 2：现场勘查（15-20分钟）**
- 与客户一起走遍所有空间
- 了解客户的使用需求和痛点
- 记录现有状况（水电位置、结构问题等）
- 询问特殊需求（收纳、功能等）

**Step 3：详细测量（30-60分钟）**
- 逐一测量每个空间：
  - 长、宽、高
  - 门窗位置和尺寸
  - 梁柱位置
  - 水电点位
  - 现有设备位置（空调、热水器等）
- 拍照记录（每个空间至少4个角度）
- 记录特殊结构（飘窗、凹位等）

**Step 4：现场沟通（10-15分钟）**
- 确认测量数据
- 讨论初步想法
- 了解材料偏好
- 确认预算范围
- 说明后续流程

## 💰 阶段三：报价生成（Quote Generation）

### 3.1 报价准备

**所需资料：**
- [ ] 测量记录和照片
- [ ] 客户需求清单
- [ ] 预算范围
- [ ] 材料偏好
- [ ] 特殊要求

### 3.2 报价编制流程

**Step 1：空间规划（1小时）**
- 根据测量数据绘制平面图
- 规划空间功能布局
- 确定改造范围

**Step 2：材料选型（1-2小时）**
- 根据预算和风格选择材料
- 参考 quote-reference.md 价格标准
- 准备2-3个材料方案（基础/标准/高端）

**Step 3：费用计算（1-2小时）**
- 材料费用明细
- 人工费用计算
- 增值服务费用（如适用）
- 使用标准报价单模板（quote-template.md）

**Step 4：方案整理（30分钟）**
- 整理报价单
- 准备案例参考
- 制作简单的方案说明

### 3.3 报价原则

| 原则 | 说明 |
|------|------|
| **透明性** | 所有费用明细列出，无隐藏收费 |
| **完整性** | 覆盖所有工程内容，避免后期增项 |
| **合理性** | 价格符合市场水平，有竞争力 |
| **灵活性** | 提供不同档次选择，适应不同预算 |
| **准确性** | 基于实际测量数据，误差控制在5%内 |

### 3.4 报价档次设置

每个项目应提供 **2-3个报价档次**：

| 档次 | 定位 | 材料标准 | 利润率 |
|------|------|---------|:------:|
| **经济档** | 满足基本需求 | 国产/本地品牌 | 15-18% |
| **标准档** | 性价比最优 | 知名品牌 | 18-22% |
| **高端档** | 品质优先 | 进口/高端品牌 | 22-25% |

### 3.5 报价生成后处理

- [ ] 检查报价准确性
- [ ] 保存报价文件（PDF格式）
- [ ] 录入CRM系统
- [ ] 提交内部审核

## ✅ 阶段四：内部审核（Review）

### 4.1 审核流程

```
报价员编制 → 项目经理审核 → 销售主管审批 → 报价定稿
```

### 4.2 审核要点

**项目经理审核（必查项）：**

| 序号 | 审核内容 | 检查标准 |
|:----:|---------|---------|
| 1 | 测量数据准确性 | 与现场记录核对 |
| 2 | 材料数量计算 | 面积计算无误 |
| 3 | 材料单价 | 符合采购价格标准 |
| 4 | 人工费用 | 符合工时标准 |
| 5 | 工期估算 | 合理可行 |
| 6 | 利润率 | 符合公司标准 |
| 7 | 特殊需求覆盖 | 客户需求全部体现 |

**销售主管审批（必查项）：**

| 序号 | 审核内容 | 检查标准 |
|:----:|---------|---------|
| 1 | 价格竞争力 | 对比市场水平 |
| 2 | 报价完整性 | 无遗漏项 |
| 3 | 条款合规性 | 合同条款完整 |
| 4 | 客户定位匹配 | 档次选择合理 |
| 5 | 风险评估 | 潜在风险识别 |

### 4.3 审核结果处理

| 结果 | 处理方式 | 时限 |
|------|---------|:----:|
| **通过** | 进入发送阶段 | 当天 |
| **修改** | 返回报价员修改后重新提交 | 1天内 |
| **驳回** | 说明原因，重新编制 | 2天内 |

## 🔄 阶段六：客户跟进（Follow-up）

### 6.1 跟进时间表

| 时间节点 | 跟进方式 | 跟进内容 |
|---------|---------|---------|
| **发送当天** | WhatsApp | 确认收到，询问初步反馈 |
| **第1天** | 电话 | 了解是否有疑问，邀请方案讲解 |
| **第3天** | WhatsApp | 分享相关案例，保持联系 |
| **第7天** | 电话 | 了解决策进度，解决顾虑 |
| **第14天** | WhatsApp/电话 | 了解是否需要调整方案 |
| **第21天** | 电话 | 报价即将到期提醒 |
| **第30天** | 电话/电邮 | 报价到期，是否续期或重新报价 |

### 6.2 跟进话术

**第1天跟进（电话）：**
```
您好 [客户姓名]，我是 KV Premium Reno 的 [姓名]。
昨天给您发了报价方案，请问有收到吗？
有没有什么疑问或者需要我详细说明的地方？
我们也可以安排一次面对面的方案讲解，让您更清楚了解整个装修计划。
请问您什么时候方便？
```

**第3天跟进（WhatsApp）：**
```
Hi [客户姓名]！😊
之前给您看的项目是我们类似户型的案例，您可以参考一下装修效果。
[附案例照片]

有任何问题随时问我哦！
```

**第7天跟进（电话）：**
```
您好 [客户姓名]，我是 [姓名]，想跟进一下报价方案的考虑情况。
请问您对方案有什么想法呢？
如果有需要调整的地方，我们可以根据您的需求重新优化。
```

**第14天跟进：**
```
您好 [客户姓名]！
距离报价发出已经两周了，想了解一下您的决定。
如果预算方面有变化，我们可以提供不同档次的方案。
或者如果您还在比较其他公司，也欢迎把我们的方案作为参考。
```

**第21天提醒：**
```
您好 [客户姓名]！
温馨提醒：您的报价将在9天后到期。
如果您还在考虑中，我们可以延长报价有效期。
或者根据您的最新需求，重新出具一份报价。
请问需要怎么处理呢？
```

### 6.3 常见客户异议处理

| 异议类型 | 应对策略 |
|---------|---------|
| **价格太贵** | 强调合同锁价+延误赔偿的价值，拆分报价说明每项费用，提供经济档方案 |
| **需要比较** | 提供对比清单，强调差异化优势（锁价、赔偿、数字化追踪） |
| **预算不够** | 提供分期付款方案，推荐经济档，建议分阶段装修 |
| **工期太长** | 说明加急服务，解释质量与工期的平衡 |
| **需要和家人商量** | 提供家庭决策资料，邀请全家参与方案讲解 |
| **对材料不了解** | 安排材料展厅参观，提供材料样品 |

### 6.4 跟进结果处理

| 结果 | 后续动作 | CRM状态 |
|------|---------|:-------:|
| **接受报价** | 安排签约，准备合同 | Won |
| **需要修改** | 修改后重新报价 | 修改中 |
| **暂时搁置** | 加入长期跟进名单 | 搁置 |
| **拒绝** | 记录原因，礼貌结束 | Lost |
| **无回复** | 继续跟进至30天 | 跟进中 |
