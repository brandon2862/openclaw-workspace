## 1. 受众定位策略概述

### 定位策略框架

```
┌─────────────────────────────────────────────────────────┐
│                    核心受众 (Core Audience)               │
│  ┌───────────────────┐    ┌───────────────────┐         │
│  │  华人 T20 受众群   │    │  马来 M40-T20 受众群 │        │
│  │  (35-55岁)        │    │  (25-45岁)         │        │
│  └───────────────────┘    └───────────────────┘         │
├─────────────────────────────────────────────────────────┤
│                    自定义受众 (Custom Audience)           │
│  ┌──────┐ ┌──────┐ ┌──────┐ ┌──────┐ ┌──────┐         │
│  │网站访客│ │视频  │ │表单  │ │互动  │ │客户  │          │
│  │      │ │观看者│ │提交者│ │用户  │ │列表  │          │
│  └──────┘ └──────┘ └──────┘ └──────┘ └──────┘         │
├─────────────────────────────────────────────────────────┤
│                    相似受众 (Lookalike Audience)          │
│  ┌───────────────────┐    ┌───────────────────┐         │
│  │  高价值客户相似     │    │  网站访客相似       │        │
│  │  (1%-3%)          │    │  (1%-5%)           │        │
│  └───────────────────┘    └───────────────────┘         │
└─────────────────────────────────────────────────────────┘
```

### 三层受众定位体系

| 层级 | 受众类型 | 预期规模 | 转化率 | 预算占比 |
|------|---------|---------|--------|---------|
| **L1 冷受众** | 核心受众（兴趣/人口统计） | 50万-100万 | 低（1-2%） | 40% |
| **L2 暖受众** | 自定义受众（互动/访问） | 5千-2万 | 中（3-5%） | 35% |
| **L3 热受众** | 相似受众 + 高意向群体 | 1万-5万 | 高（5-10%） | 25% |

| 受众名称 | 条件 | 保留天数 | 用途 |

## 5. 受众排除策略

### 必须排除的受众

| 排除受众 | 排除原因 | 操作方式 |
|---------|---------|---------|
| **已签约客户** | 避免浪费预算 | 上传客户列表排除 |
| **已咨询客户（短期）** | 正在跟进中，避免干扰 | 自定义受众排除 |
| **员工/内部人员** | 避免内部点击浪费 | 上传员工邮箱排除 |
| **竞争对手区域** | 非目标市场 | 地理位置排除 |

### 建议排除的受众

| 排除受众 | 排除条件 | 排除原因 |
|---------|---------|---------|
| **低质量线索** | 表单提交但电话无法接通 | 提高线索质量 |
| **重复用户** | 30天内多次提交表单 | 避免重复跟进 |
| **非目标年龄** | 年龄 <25 或 >60 | 不符合目标客群 |

## 8. 受众定位检查清单

### 投放前检查

- [ ] 地理位置是否覆盖目标区域
- [ ] 年龄范围是否符合目标客群
- [ ] 语言设置是否正确
- [ ] 兴趣关键词是否相关
- [ ] 排除列表是否更新
- [ ] 预算分配是否合理
- [ ] 受众规模是否适中（10万-50万为佳）

### 投放中监控

- [ ] 每日检查各受众组表现
- [ ] 监控广告频率（不超过5）
- [ ] 检查线索质量
- [ ] 记录高效受众特征

### 投放后复盘

- [ ] 分析各受众组CPL差异
- [ ] 识别高效受众特征
- [ ] 更新相似受众种子
- [ ] 优化下一轮受众策略
